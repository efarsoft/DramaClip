## 情感TTS模型调研 + CosyVoice/Qwen3-TTS 集成

### 调研结论
- CosyVoice V3（DashScope API）— 情感控制最强，`dashscope.audio.tts_v2.SpeechSynthesizer`，仅 V3 + Instruct音色支持情感
- Qwen3-TTS instruct-flash — 支持 `instructions` + `optimize_instructions` 参数做情感控制
- 两个都已通过 DashScope API 集成（无需本地部署大模型）

### 实现改动（4文件）
1. **config.example.toml** — 新增 `[cosyvoice]` 配置段（api_key/model/voice/instruction）
2. **config/config.py** — 加载并保存 `cosyvoice` 配置变量
3. **voice.py** — 新增 `cosyvoice_tts()` + `parse_cosyvoice_voice()`；增强 `qwen3_tts()` 支持 instructions 情感参数
4. **audio_settings.py** — 新增 CosyVoice 完整 UI 面板（模型/音色/情感预设）；增强 Qwen3 面板（情感开关+预设模板）

### 关键技术细节
- CosyVoice 用 `dashscope.audio.tts_v2.SpeechSynthesizer`（不是 MultiModalConversation）
- Qwen3-TTS instruct 模型自动切换：enable_instruct=True 时用 qwen3-tts-instruct-flash
- CosyVoice V3 情感音色：longanyang（男）、longanhuan（女）、longhuhu_v3（童声）
- Qwen3 TTS voice 前缀 `qwen3:`，CosyVoice 前缀 `cosyvoice:`
